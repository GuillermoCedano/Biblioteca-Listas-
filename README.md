Descripción del Proyecto:
Este proyecto es un programa hecho en C# para gestionar una biblioteca usando listas. Permite registrar libros, buscarlos por título y mostrar todos los libros que hay.

Objetivo:
Aprender a usar clases y listas en C# para manejar información de forma organizada.

Funcionalidades:
Registrar libros
Buscar libros por título
Mostrar lista de libros
Salir del programa

El programa usa una clase llamada Libro y otra Biblioteca que maneja una lista de libros. Todo se hace en consola con un menú simple.
